# Installation

## Create a new project

1. Clone the repository:

```bash
git clone https://github.com/hlassiege/bloggr.git bloggr
```

2. Install the dependencies from the root of the project.

::code-group

  ```bash [npm]
  npm install
  ```

  ```bash [yarn]
  yarn install
  ```

  ```bash [pnpm]
  pnpm install --shamefully-hoist
  ```

::

3. Run the `dev` command to start Bloggr in development mode:

::code-group

```bash [npm]
npm run dev
```

```bash [yarn]
yarn dev
```

```bash [pnpm]
pnpm run dev
```
